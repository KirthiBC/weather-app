# Weather App

A real-time weather application that fetches live data for any city.

## Features
- Real-time weather data for any searched city
- Clean responsive UI
- Weather icons for different conditions

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **API:** OpenWeatherMap REST API

## Setup
1. Get a free API key from openweathermap.org
2. Replace `YOUR_OPENWEATHERMAP_API_KEY` in logic1.js with your key
3. Open index.html in browser
